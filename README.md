# MobileNet-V2

This repository contains the official source code, simulation files, documentation, and reproducibility materials for the manuscript submitted to *The Visual Computer*:

**"Hybrid MobileNet-BiLSTM Framework for Efficient Real-Time Violence Detection in Surveillance Videos"**

---

## 1. Overview

This repository provides the implementation and simulation materials used in the manuscript. The purpose of this repository is to enable readers and reviewers to reproduce the experiments, evaluate the results, and understand the implementation details of the proposed method.

The repository includes:

- Source code used in the study
- Simulation file used for generating or evaluating experimental results
- Dataset access information
